# Product Requirements Document

## Goodwill Public School Patti Pratapgarh Website

---

## 1. Product Overview

**Product Name:** School Website CMS
**School Name:** **Goodwill Public School Patti Pratapgarh**
**School Type:** Indian School (CBSE Pattern)
**Developed By:** **DurgaAI Solutions**

**Frontend Technology:** React Js
**Backend Technology:** Node Js With Express
**Database:** MongoDB
**Admin Panel:** React Based Admin Dashboard
**Hosting Environment:** Cloud Based (AWS / Vercel / Netlify)

### Product Vision

To Build A Modern, Trustworthy, And Emotionally Appealing School Website That Reflects Academic Excellence, Moral Values, And Discipline While Being Fully Manageable Through An Admin Panel Without Developer Dependency.
